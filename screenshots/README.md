## Screenshot Guidelines

### Recommended Settings
- **Device**: Android device RF8M73ELTPT (or any test device)
- **Orientation**: Portrait mode
- **Resolution**: Native device resolution
- **Format**: PNG (preferred) or JPG

### Tips for Best Screenshots
1. Ensure good lighting and clear visibility
2. Capture full screen (no partial views)
3. Show relevant UI elements clearly
4. Use actual tracking data (not empty screens)
5. For heatmap screenshots, ensure circles are visible and distinct

### How to Take Screenshots

**Android**:
- Press Power + Volume Down simultaneously
- Screenshots saved to Pictures/Screenshots folder

**iOS**:
- Press Side Button + Volume Up simultaneously
- Screenshots saved to Photos app

### File Naming Convention
- Use lowercase with hyphens
- Match the exact names listed above
- Use PNG format for best quality
